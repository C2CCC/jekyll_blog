而实际上，PropDef<Prop> 才是我们传进去的类型，这边是 通过 PropDef<Prop> 推导出了 Prop

这边的重点就是 通过 PropDef<Prop> 推导出了 Prop

我们写 Vue.extend 的时候定义的是 PropDef<Prop> ，而 Prop 则是通过 TypeScript 推导出来的，因此我们在其他地方才能获得正确的类型。

结论
